FINOM Integrations

FINOM is a financial service that combines a banking business account, invoicing, and accounting software integration

Triggers

Incoming Invoice Was Paid
Triggers when the incoming invoice status changed to Paid or Partially Paid.
New Incoming Invoice
Triggers when a new incoming invoice received.
New Invoice Created
Triggers when the invoice is created.
Outgoing Invoice Was Paid
Triggers when the outgoing invoice status changed to Paid or Partially Paid.

Actions

Create a Contact
Allows to create a contact which can be used for creating invoices.
Download Incoming Invoice as JSON
Allows to use the JSON version of the incoming invoice wherever is needed
Download Incoming Invoice as PDF
Allows to use the PDF version of the incoming invoice wherever is needed
Download Outgoing Invoice as JSON
Allows to use the JSON version of the outgoing invoice wherever is needed
Download Outgoing Invoice as PDF
Allows to use the PDF version of the outgoing invoice wherever is needed
Mark an Outgoing Invoice as Paid
Allows to mark an existing invoice as paid by a customer.
Mark an Incoming Invoice as Paid
Allows to mark an existing incoming e-invoice as paid.
