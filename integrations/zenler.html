Zenler Integrations

Zenler is a super easy platform that lets you start your own branded website. It allows you to make money teaching your own courses online.

Triggers

Course Complete
Triggers when a student completes a course.
Lesson Complete
Triggers when a student completes a lesson in a course.
New Live Class Registration
Triggers when a student registers zenler live class.
New Live Interactive Webinar Registration
Triggers when a user registers zenler live interactive webinars.
New Sale
Triggers when a student enrolls in a course regardless of whether it's a free or a paid course.
New User
Triggers when a new user signs up on the site.
New Funnel Subscription
Triggers when a user subscribes to a funnel

Actions

Create User
Creates a new user in your Zenler site.
Enroll User
Enrolls the user into your Zenler course.
Register Live Class
Register the user into your Zenler Live Classes.
Register Live Webinar
Register the user into your Zenler Live Webinars.
Register Live Interactive Webinar
Register the user into your Zenler Live Interactive Webinars.
Subscribe Funnel
Subscribes the user into your Zenler Marketing Funnels.
Unenroll User
Unenrolls the user from your Zenler course.
