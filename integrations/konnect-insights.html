Konnect Insights Integrations

Konnect Insights is a social listening and analytics platform that gives you data from social platforms and web. Data is fetched based on keywords and social profiles. Get the power of social Listening with this integration.

Triggers

Get Clusters
Triggers when a new cluster is created.
Get Groups
Triggers when a new group is created.
Get Messages by Cluster
Triggers when a new ticket is created in the cluster.
Get Messages by Profile
Triggers when a new ticket is created linked to profile.
Get Messages by Topic
Triggers when new tickets are created in the topic.
Get Profiles
Triggers when new profile is created.
Get Topics
Triggers when a new topic is created.
