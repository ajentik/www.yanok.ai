YouCanBook.Me Integrations

Take bookings online. YouCanBook.me links to your calendar to allow your customers to book your services via the web.

Triggers

Updated (Cancelled) Booking
Fires when you or your user cancels the booking through our system
New Booking
Fires every time someone makes a booking with you through YouCanBook.Me
Updated (Rescheduled) Booking
Fires whenever a booking is rescheduled through our system

Actions

Create Voucher
Issue a voucher or coupon for one of your profiles. Trigger this action with the correct code whenever you sell a booking voucher to someone. If you have set up vouchers in that profile's settings, the user will then be able to make a booking using it.
