Rebump Integrations

Rebump is a Gmail extension that automates email follow-up.

Triggers

Bump Sent
Triggers when a bump on a message has been sent.
Message Expired
Triggers when all bumps on a sequence for a given message have been sent and no reply have been received.
New Message
Triggers when a new message has been added for tracking.

Actions

Track Message
Starts tracking an existing email message in Rebump
Add Personalization Token
Adds a new personalization token
Cancel Message
Cancels a message and stops tracking
Pause Message
Pauses rebumping of a message
Reply Message
Flags a message as replied
