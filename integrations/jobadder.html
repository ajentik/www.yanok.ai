JobAdder Integrations

JobAdder is a global recruitment platform built for agency and in-house recruiters. JobAdder creates a massive competitive advantage for recruiters by allowing them to spend less time using technology, and more time building face-to-face relationships.

Triggers

Entity Added to Folder
Triggers when selected entity is added to a folder.
Job Ad Expired
Triggers when a job ad has expired on a job board. Will trigger multiple times if posted on multiple job boards
New Retained Invoice Sent
Triggers when a retained invoice is sent
Applications - New or Status Changed
Triggers when you receive a new Job Application and/or its status is changed.
Candidates - New or Status Changed
Triggers when a new Candidate record is created and/or its status is changed.
Companies - New or Status Changed
Triggers when a new Company record is created and/or its status is changed.
Contacts - New or Status Changed
Can trigger when a new Contact record is created and/or its status is changed.
New Interview Scheduled
Triggers when you have scheduled an interview.
New Job Ad Posted
Triggers when a new job ad is posted on a job board. Will trigger multiple times if posted on multiple job boards
Job Orders - New or Status Changed
Triggers when a new Job Order is created and/or its status is changed.
Placements - New or Status Changed
Triggers when a new Placement record is created and/or its status is changed.
Entity Removed From Folder
Triggers when selected entity is removed from a folder.
