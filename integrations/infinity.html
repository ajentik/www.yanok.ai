Infinity Integrations

Infinity is a flexible project management platform that lets you organize anything, your way.

Triggers

New Board
Triggers when a new Board is created or you are added to a board.
Comment Created
Triggers when a new comment is created.
New Folder
Triggers when a new Folder is created.
Item Created
Triggers when a new item is created.
Item Updated
Triggers when an item is being updated.
New Member
Triggers when a new Member is created

Actions

File Upload
Uploads a file from URL or file upload to the workspace given.
Create a New Comment
Creates a new comment on a specific item.
Create a New Item
Creates a new item in a specific workspace, board and folder.
Create a New Folder
Creates a new folder in a specific workspace, board, and optionally as a child of another folder.
Update an Existing Item
Updates an existing item in a specific workspace, board and folder, based on the item ID.
Find a Board
Find a board in one of your workspaces
Find a Folder
Find a folder in one of your boards
Find an Item
Find an item in one of your boards or their folders
Find a Member
Find a member of one of your workspaces
