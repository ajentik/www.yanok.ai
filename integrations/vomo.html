VOMO Integrations

VOMO is a volunteer engagement platform that helps organizations recruit, manage, and reward their volunteers.

Triggers

Kiosk
Triggers when a check-in/check-out timestamp occurs.
Membership
Triggers when a VOMO User is created/updated/deleted as a Member of your VOMO Org based on selected actions.
Participation
Triggers when a User Participation is created/updated/deleted on one of your VOMO Projects based on selected actions.
Project
Triggers when a VOMO Project is created/updated/deleted based on selected actions.

Actions

Create Draft Projects
Creates a Draft Project in VOMO that you can publish from your Admin Dashboard.
Update Draft Projects
Updates a Draft Project in VOMO that you can publish from your Admin Dashboard.
Find Organizer
Finds a organizer based on email.
Find Draft Projects
Finds a draft project.
Find Projects
Finds a project.
