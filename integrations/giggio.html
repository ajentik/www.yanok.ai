Giggio Integrations

Giggio is a web-based system for entertainers to manage their bookings, invoices and payments.

Triggers

Act Contract Confirmed
Triggered when an act contract is approved in Giggio
Act Contract Emailed
Triggered when an act contract is emailed from the Giggio events page
Booking Form Confirmed
Triggered when a booking form is approved in Giggio
Booking Form Emailed
Triggered when a booking form is emailed from Giggio
Client Contract Confirmed
Triggered when a client contract is approved in Giggio
Client Contract Emailed
Triggered when a client contract is emailed from the Giggio events page
Event Edited
Triggered when an existing event is saved in Giggio
New Enquiry
Triggers when a new enquiry is received in Giggio.
New Event
Triggered when an event is saved for the first time in Giggio
Performer Added or Updated on Event
Triggers when a performer gets added/updated/deleted on a Giggio event.
Client Added or Updated
Triggers when a client is added or updated.

Actions

Create Client
Creates a client. If a client with supplied email address exists, it will be updated.
Create Enquiry
Creates an enquiry
Create Perfomer
Create Performer in Giggio.
