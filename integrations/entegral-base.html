Entegral Base Integrations

Base is an online property, listing syndication, contacts and document management system for estate agents.

Triggers

New Agent
Triggers when a new agent is added in the Base.
New Contact (Any Contact)
Triggers when any new contact (office shared or private) is added in the Base.
New Contact (Office Shared Contact)
Triggers when a new office shared contact is added in the Base.
Contact Updated (Office Shared Contact)
Triggers when office shared contact is updated in the Base.
Contact Updated (Any Contact)
Triggers when any contact is updated in the Base.
New Lead
Triggers when a new lead is received in the Base. Lead will be access based on Primary API user's access.
New Property
Triggers when a new property is created in Base.
Property Updated
Triggers when a property is updated in Base.

Actions

Save Contact Activity
Add new contact activity(calendar event) in the Base
Send SMS to Contact
Sends an SMS to an existing contact, limited to 140 characters.
Save Contact
Add new contact in Base
Find Agent
Finds an Agent
Find Property
Finds a Property
