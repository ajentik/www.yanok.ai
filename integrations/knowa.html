Knowa Integrations

Knowa is a board governance tool to meet, collaborate and access board information in one place

Triggers

Message Resolved
Triggers when a voting / must read message is resolved (no more actions allowed)
New Attached Document in Discussions
Triggers when a new document attached to a message in discussion you are a member of, and message was posted.
New Discussion
Triggers when a new discussion is created in a group you are a member of.
New / Updated File in Group Documents
Triggers when a new file was uploaded
Added to Group
Triggers when you are added to a group.
New Message
Triggers when a new message is posted in a discussion you are a member of.
New User in Space
Triggers when a new user is added to a space you are a member of.

Actions

Create Discussion
Creates a new discussion. (Space Member/Admin only)
Archive Discussion
Archives an existing discussion. (Group Member/Admin only)
Update Discussion
Updates an existing discussion. (Group Member/Admin only)
Create Group
Creates a new group. (Space Admin only)
Post New Message
Creates a new message in discussion.
Create Space
Creates a new space. (Org Admin only)
