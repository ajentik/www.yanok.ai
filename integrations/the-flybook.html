The Flybook Integrations

The Flybook is a reservation software for the tourism industry. It handles everything from scheduling to payments to online booking and more.

Triggers

Front End Booking
Triggers when a reservation is created from an external source (Front End, OTA, etc..).
Guest Feedback Submitted
Triggers when a guest provides feedback via the internal feedback system.
Inquiry Form Submitted
Triggers when an inquiry form is submitted.
Scheduled Activity Exported
Triggers when a user manually chooses to export a scheduled activity.
Waiver Signed
Triggers when a waiver is signed and filed.
