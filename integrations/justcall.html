JustCall Integrations

JustCall helps sales and support teams to connect easily with their leads & customers. We provide a cloud based service that allows you to make calls and send text messages.

Triggers

New Appointment
Triggers when a new appointment is scheduled via JustCall Appointment Scheduler (Public Calendar)
Call Ended
Triggers when a call ends and gets logged in your JustCall call logs. (For voicemails use New Voicemail trigger)
New Contact
Triggers when a new contact is added to your JustCall address book.
New Text
Triggers when a new text is sent from (or received on) your JustCall numbers
New Voicemail
Triggers whenever there is a new voicemail
Call Updated
Triggers when you add/update notes and disposition codes for a particular call

Actions

Add to Sales Dialer Campaign
This action allows you to add a contact to an existing Sales Dialer Campaign.
Check for Text Replies
Helps you check if you have received a reply from a number on one or across all of your JustCall numbers
Create Contact
Creates a Contact
Delete Contact From Sales Dialer Campaign
This action allows you to delete a contact from an existing Sales Dialer Campaign
Send a Multimedia Text Message
Allows you to drop a multimedia text message
Schedule a Text Message
This action allows you to schedule a text message. This message goes out at a particular date and time in future.
Send a Text Message
Allows you to send a text message to any number
Update Contact
updates the contact in Justcall
Find Contact
Finds a contact
