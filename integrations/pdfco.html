PDF.co Integrations

PDF.co is a set of tools for PDF file and data extraction. Get CSV out of pdf, spreadshets, barcodes, invoices. Also can split pdf, merge pdf, convert pdf.

Triggers

Actions

Barcode Generator
Barcode generation from QR Code, PDF417, Datamatrix to Code 128 and many others
Barcode Reader
Read barcode from image or PDF document
Anything to PDF Converter
Convert URL, HTML code, document, spreadsheet, presentation, image, file to PDF
PDF to Anything Converter
Convert PDF to JPG, PNG, CSV, XML, JSON and other formats
Custom API Call
Make a custom API call to PDF.co API (for advanced users)
Document Classifier
Analyzing text of input document using AI and returns its class (invoice, order, industry etc). Can also use custom rules. Useful for quickly finding where this source document came from.
Document Parser
Parse invoices, reports, statements and other documents using templates based document parser.

Send Email With Attachment
PDF File Information Reader
Get PDF document information like form fields, pages count, page size, author, description, keywords and other information.
HTML to PDF Converter
Convert HTML or URL to with avdanced options (layout, orientation etc). Supports HTML templates (Handlebar and Mustache).
Make PDF Searchable
Turns Image or PDF into text searchable PDF where you can select and search text.
PDF Merger
Merge multiple documents into one new single PDF. Supports pdf, doc, docx, xls, jpg, png, etc.
PDF Compressor

PDF Filler
Add text, images, signatures to pdf, fill PDF form fields, create new pdf from templates.
PDF Find Table
AI powered document analysis to scan your document for tables and return the array of tables on pages with coordinates and information about columns detected in these tables.
PDF Security
