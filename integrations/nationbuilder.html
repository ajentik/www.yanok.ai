NationBuilder Integrations

NationBuilder is designed to power your organizations, movements, and campaigns.

Triggers

New Contact Made
Triggers when a certain person in your nation makes a new contact.
New Donation
Triggers when a new donation is created.
New Event
Triggers when a new event is created.
New Person
Triggers when a new person is created.
Person Updated
Triggers when a person in your nation is updated.

Actions

Add Person to List
Adds a person to a specified list.
Award Political Capital
Awards (or deducts) political capital to a person in your nation.
Create Logged Contact
Creates a contact entry against a person.
Create Donation
Creates a donation.
Create Event
Creates an event in your nation, on a site of your choice.
Create Event RSVP
Creates an event RSVP
Create/Update Membership
Creates or updates membership for a person.
Create/Update Person
Creates a person in your nation if they don't exist. Updates an existing person if they do exist.
Create Petition Signature
Add a new petition signature to a person
Delete Tag
Removes a tag from a person in your nation.
Delete Multiple Tags
Deletes multiple tags (from a list of tag names).
Create Private Note
Add a private note to a user's profile
Update Event
Updates event details
Update Membership
This action allows you to update an existing membership
Find Logged Contact
Searches for a logged contact in the People database.
Find Person
Finds a person in your nation.
Return Donations by Donor ID
This action can be used to look up a donor based off of the donor ID (signup ID or person ID).
Return Person
Finds a person using a person ID.
Find or Create Person
Finds a person in your nation.
