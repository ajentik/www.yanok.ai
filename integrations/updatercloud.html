UpdaterCloud Integrations

UpdaterCloud is a fast and flexible way to integrate web updates support with optional license key authorization to your free or premium web applications.

Triggers

Customer Deleted
Triggers when a customer is deleted.
Download Key Deleted
Triggers when a download key is deleted.
Download Key Expired
Triggers when a download key has expired.
Download Key Expiring
Triggers when a download key is about to expire.
Customer Created
Triggers when a new customer is created.
Download Key Created
Triggers when a new download key is created.
Customer Updated
Triggers when an existing customer is updated.
Download Key Updated
Triggers when an existing download key is updated.
New Version Released
Triggers when a new version is released (published).
