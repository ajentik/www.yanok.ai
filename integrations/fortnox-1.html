Fortnox Integrations

Fortnox is a cloud-based platform that meets the needs of small businesses and accounting agencies for programs and services that can manage their finances efficiently.

Triggers

Customer Created
Triggers when a new customer is created or first time we see an update on it.
Invoice Bookkept
Triggers when an invoice is bookkept.
New Article
Triggers when a new article is created.
New Order
Triggers when a new order is created. Note that a order created from an Offer may have rows but if you just created the order manually it will probably not. Please use something like "Delay by Zapier" and then fetch the order again using "Find Order" to give yourself more time to set rows and other properties.

Actions

Bookkeep Invoice
Bookkeep an existing invoice.
Create or Update Customer
Creates or updates a customer
Create Invoice
Creates an Invoice
Create Invoice Row
Creates an Invoice Row for an existing invoice.
Create Order
Creates an order
Create Order Row
Creates an Order Row for an existing Order.
Create or Update Project
Updates existing project or create a new one from given project number.
Create or Update Article
Creates or Updates Article
Create or Update Cost Center
Creates or Updates Cost Center based on Cost Center Code
Find Article
Finds a Article
Find Order
Finds a Order based on order number
