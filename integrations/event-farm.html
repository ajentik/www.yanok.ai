Event Farm Integrations

Event Farm is an enterprise event marketing platform: invitation & digital activation designed to promote and engage targeted audiences.

Triggers

Affirmative Invitation
Triggers when a guest's invitation receives an affirmative status for the selected event.
Assigned Invitation
Triggers when a guest is assigned an invitation for the selected event.
New Check-In
Triggers when a guest checks into the selected event.
Confirmed Invitation
Triggers when a guest confirms their invitation into the selected event.
Added Invitation
Triggers when a new guest is added to your selected event.
Left Behind Invitation
Triggers when an invitation is left behind for the selected event.
Purchased Invitation
Triggers when a guest purchases an ticket or invitation for the selected event.
Registered Invitation
Triggers when a guest registers for the selected event.
Removed Invitation
Triggers when a guest's invitation is removed from the selected event.
Updated Invitation
Triggers when any part of a guest's ticket / invitation is updated or changed.

Actions

Create Invitation
This call will add the information provided as a Guest in the Event Farm system and add that Guest to the Guest List for the specified Event with the provided configuration details.
