Tilkee Integrations

Use Tilkee's latest API to track how your contacts read your documents.

Triggers

New Connexion
Triggers when a new connexion is done on your projects.
New Contact Accessed Your Document
Triggers when a new contact access your document for the first time using the link with authentication.
Document Has Been Signed
Triggers when a contact signed a document through one of your access link.

Actions

Add Components
Add component to an existing project. You can add an existing item as a component to your project or create a new one and add it to your project.
Create Item
Creates an item.
Create Project
Create a project and its access links.
Update Project
Update a specific project.
Get Items
Search and list your items.
Get One Project
Get a specific project.
Get Projects
List the existing projects.
