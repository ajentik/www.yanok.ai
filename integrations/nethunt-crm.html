NetHunt CRM Integrations

NetHunt CRM is a customer relationship management system for Gmail and Google Apps.

Triggers

Deleted Record
Triggers when a record gets deleted.
New Call Log
Triggers when a new record call log is created.
New Comment
Triggers when a new record comment is created.
New File
Triggers when a new file is added to a record.
New Record
Triggers when a new record is created.
New or Updated Record (Advanced)
Triggers each time a record is created, modified, or deleted. Provides the change type, author, time, 'from' and 'to' values. Requires a new record search to obtain an up-to-date record if necessary.
New, Updated, or Deleted Record
Triggers with an up-to-date record when it is created, modified, or deleted.
New or Updated Record in Filter
Triggers when a record matching a specific filter gets created or modified.
