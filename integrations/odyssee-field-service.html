Odyssee Field Service Integrations

Odyssee Field Service is a work order management software that helps service companies around the world achieve field service excellence.

Triggers

New Contact
Triggers when the customer request all contact list.
New Company
Triggers when you want to get the last modified 20 companies.
New Invoice
Triggers when a new invoice is created
New Invoice
Triggers when a new invoice is created (An empty article_name will be filled by article_description)
New Article
Triggers when a new article is created.

Actions

Create or Update Article
Add or update article in Odyssee and link it to a pre-defined price label within Odyssee Field Service.
Create or Update Contact
Create or update contact and link it to the company/account.
Create or Update Job Locations
Create an additional job location linked to a company.
Create or Update Task
Create or update a task within Odyssee Field Service. The task can be created for any company/account linked with a contact.
Create Task Activity
Creates a Task Activity under a specific task
Create or Update Company
Allows you to Create or update an existing company (account) on your domain.
Find Company
Find a company based on Company_id
Find Contact
Finds a contact details based on email address
Find Task
Finds a Task based on the contact id
Find User
Finds a user details based on email address
Find or Create contact
Finds a contact details based on email address
