Keygen Integrations

Keygen is a key generation service that makes it easy for businesses to license and distribute their software.

Triggers

License Created
Triggers when a new license is created.
License Deleted
Triggers when a license is deleted.
License Expired
Triggers when a license has expired.
License Expiring
Triggers when a license is expiring soon.
License Renewed
Triggers when a license is renewed.
Machine Activated
Triggers when a machine is activated.
Machine Deactivated
Triggers when a machine is deactivated.
New Event
Triggers when a new event like a license validation, an activation, or an expiration occurs.
Password Reset
Triggers when a user requests a password reset.
Release Created
Triggers when a new release is created.
Release Published
Triggers when a release is published.
User Created
Triggers when a new user is created.

Actions

Checkout License
Checks out a license file.
Create License
Creates a new license according to a policy.
Create Token
Creates a new API token for a license.
Create User
Creates a new user.
Download Artifact
Creates a download link for a release artifact.
Reinstate License
Reinstates a suspended license.
Renew License
Renews a license by extending its expiration date.
Reset Password
Requests a password reset for an email.
Suspend License
Suspends a license.
Transfer License
Transfer an existing license to a different policy.
Update License
Updates a license.
Update User
Updates a user.
Find Artifact
Finds an existing release artifact by its filename or Keygen ID.
Find License
Finds an existing license by its license key or Keygen ID.
Find Release
Finds an existing release by its version, tag or Keygen ID.
Find User
Finds an existing user by their email address or Keygen ID.
