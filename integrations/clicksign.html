Clicksign Integrations

Clicksign is an electronic document signing platform that meets the requirements of integrity, authenticity and non-repudiation of Brazilian legislation.

Triggers

Triggers Event When a Event Occur.
Triggers when a event or a list of events occur.

Actions

Add Signer to Document
Request to add a signatory to a document at Clicksign. This is a POST request for / api / v1 / lists. In the body, the key attributes of the document, key of the signatory and how the signatory will sign the document must be sent.
Batches
Request to batch creation of documents in Clicksign.
Cancel Document
Request for canceling documents at Clicksign.
Create Document From Template
Request for creating documents at Clicksign via template.
Create Document
Request for creating documents at Clicksign via upload
Create Signer
Request for the creation of signatories at Clicksign.
Delete Document
Request to delete a document that has already been finalized or canceled at Clicksign. The result is permanent removal of the document from the account.
Delete Signer
Request to permanently delete a signatory
Duplicate Document
Request to duplicate a document that has already been finalized at Clicksign.
List Documents
Request to view part of the metadata of documents in Clicksign.
List Templates
Request to view list of templates in Clicksign.
List Webhooks
After registering the Webhook, it can be consulted.
Notifications Email
Request to send an e-mail message to a signatory requesting to sign a document or batch.
Notifications SMS
Request to send an e-mail message to a signatory requesting to sign a document or batch.
Notifications WhatsApp
Key available in the document's JSON used in notification requests and in the Embedded Widget. Note: It is mandatory that the signer authentication is whatsapp.
Remove Signer From Document
Request to remove a signatory from a document at Clicksign.
View Document
Request to view the metadata of documents in Clicksign.
View Signer
Request to view the metadata of the signatories at Clicksign.
Update Document
Request to change document settings at Clicksign.
