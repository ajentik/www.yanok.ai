Smartsheet Integrations

Smartsheet is a leading cloud-based platform for work execution, enabling teams and organizations to plan, capture, manage, automate, and report on work at scale, resulting in more efficient processes and better business outcomes.

Triggers

Updated Row
Triggers when a row is updated.
New Attachment
Triggers when a new attachment is added to a row.
New Comment
Triggers when a new comment is added.
New Row
Triggers when a new row is added.

Actions

Create Workspace
Creates a Workspace.
Add Attachment to Row
Adds a file attachment to a row.
Add Discussion to Row
Adds discussion to row.
Add Row to Sheet
Add a row to a sheet.
Copy Folder
Copies folder to another destination.
Copy Row
Copies row to another sheet.
Copy Workspace
Creates a copy of the specified Workspace.
Move Row
Moves row to another sheet.
New Sheet From Existing Sheet
Create a new sheet from an existing sheet.
New Sheet From Template
Create a new sheet from a template.
New Row Update
Update an existing row with new values. Requires a row ID.
Send Row
Send a row via email.
Send Sheet
Send a sheet via email (as PDF or Excel).
Share Sheet
Share a sheet.
Share Workspace
Share a workspace.
