EngageBay Marketing Integrations

EngageBay is a simple, affordable all-in-one marketing and sales software built for small businesses.

Triggers

Tag Added to Contact
Triggers when a given tags is associated with the contact.
New Company
Triggers when a new company is created.
New Contact
Triggers when a new contact is added to your account upon confirmation (has completed any applicable double opt-ins).
New Deal
Triggers when a new deal is added to your account upon confirmation.
New Event
Triggers when a new event is added to your account upon confirmation.
Get Owners
Trigger to get all owners.
New Task
Triggers when a new task is added in the EngageBay.
Get Tracks
Tigger to get all tracks.
Instant Deal Update
Instant Deal Update triggers when a deal is updated to the given deal id
New Contact in List
Triggered every time a contact is added to the specified list.
New Form Submission
Triggered every time a new form submission occurs.
Ticket Created
Triggers when a new ticket is created in enagebay
Ticket Updated
Triggers when a support ticket is updated.
New Unsubscribe
Triggers when a user unsubscribes from your emails.
Update Company
Triggers when a company is updated.
Update Contact
Triggers when a contact is updates in your account.
Update Deal
Triggers when a deal is updated in your account.
Update Task
Triggers when a task is added in the EngageBay.

Actions

Add Contact to Sequence
Adds a contact to a sequence.
Create Contact
Creates a new contact.
Create Event
Creates a new event.
Create Ticket
Created New Ticket.
Add Contact to Form
Adds a contact to a form in your account.
Create or Update Contact
Updates the contact based on the email address. Creates a new contact if one is not found.
Remove Tag From a Contact
Remove tag from a contact. A tag is a label for the identification of a contact.
Add Tag to Contact
Adds a new tag to a contact. A tag is a label for the identification of a contact.
Unsubscribe From Emails
Unsubscribes the given email address from your account.
Find Contact
Finds an existing contact.
Find or Create Contact
Finds an existing contact.
