JetBrains Space Integrations

JetBrains Space is an all-in-one collaboration solution for creative teams that includes chats, blogs, meetings, VCS, code review, CI/CD, package repositories, and issues.

Triggers

New Absence Created
Triggers when a new absence is created in Space.
New Blog Article Created
Triggers when a new blog article is created in Space.
New Public Channel Created
Triggers when a new public channel is created in Space.
New Checklist Created
Triggers when a new checklist is created in the project in Space.
New Code Review Created
Triggers when a new code review is created in the project in Space.
New Issue Created
Triggers when a new issue is created in the project in Space.
New Location Created
Triggers when a new location is created in Space.
New Meeting Created
Triggers when a new meeting is created in Space.
New Membership Created
Triggers when a new membership is created in Space.
New Package Repository Created
Triggers when a new package repository is created in the project in Space.
New Profile Created
Triggers when a new profile is created in Space.
New Project Created
Triggers when a new project is created in Space.
New Role Created
Triggers when a new role is created in Space.
New Team Created
Triggers when a new team is created in Space.

Actions

Create an Issue
Creates an issue in the specified project in Space.
Create a Location
Creates a location in Space
Create a Profile
Creates a profile in Space.
Create a Project
Creates a project in Space.
Create a Role
Creates a role in Space.
Create a Team
Creates a team in Space.
Create a ToDo Item
Creates a todo item for your user in Space.
Invite a User
Invites a user by email in Space.
Send a Comment to Code Review
Send a comment to a code review (ID of the code review required) in Space.
Send a Comment to Issue
Send a comment to an issue (ID of the issue required) in Space.
Send a Direct Message
Sends a direct message to a Space user.
Send Message to Channel
Sends a message to the chat channel in Space.
