OriginStamp Integrations

OriginStamp allows to create digital timestamps that are shared within a huge network world-wide which makes them tamper-proof.

Triggers

New Timestamp is ready
Triggers when a new Timestamp is ready and the corresponding certificate is available. It means that your hash was successfully included in one or more blockchains and its Timestamp is tamper-proof.

Actions

Create Timestamp from File
Creates a new timestamp given your custom file input like PDF document, image etc.
Create Timestamp from Text
Creates a new timestamp given your custom text input like email body, document contents etc.
