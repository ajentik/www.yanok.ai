Textline Integrations

Textline is a text messaging platform for your business. Share one or more phone numbers for receiving, managing, & responding to messages.

Triggers

New Contact Message
Triggers when a contact sends you a new message.

Actions

Import Contact Message
Imports a new message into Textline as if the contact sent it by SMS.
Send Announcement
Send an announcement to one or more contacts. This can be done by phone number(s), tag, or saved search.
Send Message
Sends a new message to a contact.
Send Survey
Send a survey to one or more contacts. This can be done by phone number(s), tag, or saved search.
Update Agent Status
Updates an Agent's availability within Textline.
Create/Update Contact
Creates or updates an entry in your Textline address book.
Find Agent
Find an agent's information by email.
Find Contact
Find a contact by phone number.
