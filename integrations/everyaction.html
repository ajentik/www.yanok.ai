EveryAction Integrations

EveryAction provides nonprofits with customer relationship management, donor management, and fundraising software tools.

Triggers

Actions

Add Activist Codes
Add Activist Codes to a contact
Add Canvass Responses
Adds a canvass response to a contact (which includes survey question responses and activist codes).
Tag a Contact
Add a Tag to an existing contact
Add Contact History
Adds a Contact History record to a contact
Add a Contribution
Adds a contribution to an existing contact
Add Note to Contact
Add a note to an existing contact
Add Survey Responses
Add Survey Responses to a contact
Create or Update Contact
Attempts to find a matching contact and updates it. If a contact record is not found, it creates it.
Create or Update Contact With a Contribution
Attempts to find a matching contact and updates it. If a contact record is not found, it creates it. It then adds a Contribution to the contact record.
Create Financial Batch
Creates a new financial batch.
Submit Data to Online Actions
Submits data to a form you have published in Online Actions.
Subscribe Email
Subscribes an Email to specific lists.
Find Financial Batch
Finds an existing open Financial Batch
Find Contact
Find a Contact in the current context
Find or Create Financial Batch
Finds an existing open Financial Batch
