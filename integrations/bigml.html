BigML Integrations

BigML is a Machine Learning platform to solve and automate Classification, Regression, Time Series Forecasting, Cluster Analysis, Anomaly Detection, and Topic Modeling tasks. BigML seamlessly transforms data into actionable models that are used as remote services or, locally, embedded into applications to make predictions.

Triggers

New Resource
Triggers when a new resource is created.

Actions

Create Anomaly Score
Calculates the anomaly score of a data instance.
Create Centroid
Find out the closest cluster to your data instance.
Create Prediction
Predict using a model, logistic regression, or deepnets.
Create Topic Distribution
Calculate all topic probabilities for a given document.
Find a Resource
Finds a resource.
