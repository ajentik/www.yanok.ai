Active Collab Integrations

Active Collab is a popular, all-in-one tool for project management, team collaboration and billing. Available both in cloud and self hosted versions.

Triggers

New Comment
Triggers when a comment is added to an object in Active Collab.
New Project
Triggers when new project is created.
New Task
Triggers when a new task is created.
Task Closed
Triggers when a task is closed in Active Collab.
New Task List
Triggers when a new task list is created in a project in Active Collab.
Task Moved
Triggers when a task is moved to another task list.
Time Tracked
User has tracked time on project or task.

Actions

Add Comment to Task
Creates a new comment in selected task.
Create Company
Creates a new company.
Complete Project
Completes a project.
Complete Task
Completes a task.
Create Project
Creates a new project.
Create Task
Creates a task.
Create Discussion
Creates a new discussion in a project.
Add Comment to Discussion
Adds a new comment to discussion.
Invite User to Project
Invites a new user to a project.
