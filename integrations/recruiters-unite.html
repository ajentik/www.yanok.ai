Recruiters*Unite Integrations

Recruiters*Unite is a smart search engine dedicated to CV-Search. Use big data and AI-technology to manager and search your talent pool.

Triggers

Download Original CV File
Triggers when a original file was downloaded.
Drupal Updater
Triggers when a drupal data are updated.
New Formated CV
Triggers when a new formatted CV was created as pdf or doc-file
New or Updated Candidate Profile
Triggers when a candidate profile was created or updated in Recruiters*Unite system
Statistics Generator
Triggers when a Recruiters*Unite application trigger an event to generate daily statistics.
