Azure DevOps Integrations

Azure DevOps is where teams manage, develop, and deliver software in the cloud. Azure DevOps provides free, private Git project hosting for up to 5 users, project and test management tools, build and CI, and team collaboration. Connect using the development tools of your choice, including Visual Studio, Eclipse, Xcode, and more.

Triggers

Completed Build
Triggers when a build completes.
Code Check-In (TFVC)
Triggers when code is checked into Team Foundation version control.
Code Push (Git)
Triggers when code is committed and pushed to a Git repository on Visual Studio Team Services.
New Work Item Comment
Triggers when a work item is commented on.
New Work Item
Triggers when a work item is created.
Update Work Item
Triggers when a work item is updated.

Actions

Trigger Build
Queues a new build.
Create Work Item
Create a new work item.
Add Link to Work Item
Add a link to a work item.
