Paystack Integrations

Paystack helps African merchants accept one-time and recurring payments online with a modern, safe, and secure payment gateway.

Triggers

Abandoned Transaction
Triggers when a transaction is abandoned
Cancelled Subscription
Triggers when there's a an existing subscription is cancelled by you or the customer.
New Customer
Triggers when a new customer is created.
New Chargeback
Triggers when a chargeback is logged against your account and awaiting your feedback
New Invoice
Triggers when an invoice is created or paid.
New Order
Triggers when a new order is made on your account
New Payment Page Transaction
Triggers when there's a new successful or failed transaction from a Payment Page
New Refund
Triggers when a refund happens on your account
Subsciption Renewal
Triggers when there's a new successful or failed subscription renewal.
New Subscription
Triggers when there's a newly created subscription.
New Transaction
Triggers when there's a new successful or failed transaction.
New Transfer
Triggers when a new transfer is queued, or an existing transfer resolves successfully or fails.

Actions

Create Refund
Refunds a transaction.
Create Subscription
Creates a new subscription
Push Invoice to Terminal
Send an invoice from your application to the Paystack Terminal for processing
Send Simple Invoice
Send a payment request to your customers
Update Order
Updates the status of an order.
Check Terminal Status
Check the availiability of a Terminal before sending an event to it.
Find Customer
Find a customer record using their email address
Find Order
Find a order record using the Order ID
Find Transaction
Find a transaction record using the reference
Find Transfer
Find a transfer record using the transfer code
Get Customer Totals
Get a customer's total spend on your Paystack business
Resolve Account Number
Retrieve the full name tied to a Nigerian account number
