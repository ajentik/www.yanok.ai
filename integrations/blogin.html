BlogIn Integrations

BlogIn is the beautifully simple internal blog and knowledge sharing platform for teams of all sizes.

Triggers

New Post
Triggers when a new post is published on BlogIn.
New User
Triggers when a new user joins BlogIn.
Post Updated
Triggers when published post has been updated.
User Updated
Triggers when a user profile on BlogIn is updated.

Actions

Create New Draft
Create (but do not publish) a new post on BlogIn.
Publish New Post
Create and publish a new post on BlogIn.
