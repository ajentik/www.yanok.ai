viewneo Integrations

viewneo is a proprietary, cloud-based Digital Signage Software.

Triggers

Get Offline Device
Triggers when a device goes offline. Will not work by using update interval "realtime"!
Offline Device Group
Triggers when a complete device group is offline. Will not work by using update interval "realtime"!

Actions

Add Image File
Add a image file to a selected playlist.
Edit Text Elements
Edit your text elements of a selected slide.
Create New Slide
Create and edit a new slide from template (viewneo template plugin is required).
Change Live Ticker Text
Changes the current live ticker text of a selected device group.
Change Playlist
Changes the current playlist of an selected device group.
Toggle Live Ticker
Toggle live ticker on or off for a selected device group.
