Mojo Integrations

Mojo is a Real Estate prospecting solution which includes an easy to use CRM, Power Dialer and Real Estate lead service.

Triggers

New Contact in Group
Triggers when a contact is added to a group in Mojo (contact sync is not available for contacts within a previous integration).
New Contact
Triggers when a new contact is created (through the create contact form in Data & Dialer section).
Contact Number Marked as "Bad Number"
Triggers when a contact number is marked as "Bad Number" in Mojo.
Contact Updated
Triggers when a contact is updated. Specifically when a basic field (like name, address, etc), misc field (Listing Status, etc), note or information field (Last Call Result, etc) is changed in Mojo.
New Activity
Triggers when a new activity is created.
New Note
Triggers when a new note is created in Mojo (through the contact screen only).
Note Updated
Triggers when a note is updated in Mojo.

Actions

Create Contact in Calling List
Create Contact in Mojo and assign to calling list.
Create Contact in Group
Create a contact in Mojo and assign to one or more groups.
Create Activity
Creates a calendar activity in Mojo.
Update Contact
Update Contact in Mojo. A Contact_ID value is necessary. It can be retrieved from a New Contact, New Contact in Group or Contact Updated trigger or from the Create Contact in Group Action.
