Teachmore Integrations

Create and sell online courses with your own fully-featured mobile apps and website. Don't miss out on sales for not having mobile apps!

Triggers

Course Completed
Triggers when a user mark a course as completed on your Academy app or website.
Item Comment
Triggers when a user comments on Item from your app or website.
Item Completed
Triggers when a user completes an item
Item Reply
Triggers when a user replies to a comment of course's item in the app or website.
New Rating
Triggers when a user rate a course/collection from the app or website
New User
Triggers when a new user signs up on your Academy app or website.
Order Created
Triggers when order created regardless of paid or free product
Order Processed
Triggers when order processed on server for paid products
Post Comment
Triggers when a user comments on a post form the app or website
Post Reply
Triggers when a user replies to a comment of post in the app or website
Profile Updated
Triggers when a user updates profile from the app or website

Actions

Create User
Creates a user in your academy.
Enroll User
Enrolls a user in the course/collection. (Create one when user does not exist)
Successful Order
Creates successful order and triggers an email
Unenroll User
Unenrolls user from the selected course/collection
Find User
Finds a user from your academy by email or mobile number
