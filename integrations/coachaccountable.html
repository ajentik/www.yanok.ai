CoachAccountable Integrations

CoachAccountable is a platform to support and structure coaching relationships between coaches and their clients.

Triggers

New Client
Triggers when a new client is created.
New Coach
Triggers when a new coach is created.
New Invoice
Triggers when a new client invoice is created.
New Invoice Payment
Triggers when a new payment is posted for an invoice.
New Session
Triggers when a new Session is marked complete.

Actions

Add New Action
Adds an action for a specific client.
Mark Action Done
Marks an existing client Action done.
Activate Client
Activates a Client.
Add New Client
Adds a new Client.
Deactivate Client
Deactivates a Client.
Add Client to Course
Adds a Client as a participant in one of your Courses.
Add New Invoice
Adds a new invoice.
Add Invoice Payment
Posts a payment to a Client Invoice.
