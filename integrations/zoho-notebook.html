Zoho Notebook Integrations

Notebook is a beautifuly simple note taking app. Take notes, make lists, record audio, capture moments and sync your notes across devices.

Triggers

New Audio Card
Triggers when a new Audio Card is created.
New Note Card in Notebook
Triggers when a note card is created in a specific notebook.
New Checklist Card
Triggers when a new Checklist Card is created.
New File Card
Triggers when a new File Card is created.
New Photo Card
Triggers when a new Photo Card is created.

Actions

Create Audio Card
Creates a new Audio Card.
Create File Card
Creates a new File Card.
Create Notebook
Creates a new notebook.
Create Text Card
Creates a new Text Card.
Create Photo Card
Creates a new Photo Card.
Read Note Card as HTML
Read an existing Note Card content as HTML. Except File Cards, all other Note Card types are supported.
Read Note Card as Plain Text
Read an existing Note Card as plain text (without any styles or file attachments, if any). Audio Cards , Photo Cards and File Cards are not supported.
