Boathouse Connect Integrations

Manage your rowing organization from one place. From practice lineups to equipment to communication, Boathouse Connect can do it all

Triggers

New Equipment Issue
Triggers when a new equipment issue is logged either by post-practice surveys or by manually reporting the issue.
New Group Join Request
Triggers when a user requests to join a group with Self Join set as "No".
New User Account Charge
Triggers when a new charge is created for one of your users.
New User Account Credit
Triggers when either a user makes a payment or a credit gets added to their finance account by an admin.
New Join Request
Triggers when there is a new join request for the organization.
New Group Member
Triggers when a user becomes a member of the specified group.
New Organization User
Triggers when a new user joins your organization.
Messages Pending Review
Triggers when a email or text message needs to be approved prior to being sent.
New Practice Created
Triggers when a new practice is created.
Organization Groups
Triggers when a new group is created for the organization.

Actions

New User Pre-Authorization
Pre-Authorize a user to join your organization when they create a Boathouse Connect account.
