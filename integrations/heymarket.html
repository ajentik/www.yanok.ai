Heymarket SMS Integrations

Heymarket SMS is a business texting with customers and employees. Your team gets shared inboxes, templates, and workflows to message quickly and securely from all the major messaging apps in one place.

Triggers

Chat Started
Triggers when a new chat is started (inbound or outbound).
New or Updated Contact
Triggers when a contact is created or updated.
Incoming Message
Triggers when a message is received.
Contact Unsubscribe
Triggers when a contact unsubscribes themselves via SMS.
Incoming Phone Call
Triggers when a phone call is received.
Outgoing Message
Triggers when a message is sent.

Actions

Create or Update Contact
Creates or updates a contact.
Update List
Adds or removes contacts from a list.
Send Template Message
Sends a text message based on a template.
Send Custom Message
Sends a custom text message.
