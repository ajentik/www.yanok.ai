membermate Integrations

membermate is a platform to easily set up your own member area, including courses, memberships, community, live calls and more.

Triggers

Actions

Create New User (And Optionally Grant Access to a Product)
Create a new user / member in your memberarea. Additionally you can grant access to one or many products. If there already is a user with the given email address, no new account will be created, but access to the product(s) will be granted to the existing account.
Add Access to One or Many Products for an Existing User
Grant an existing user (by email) access to a product you created in your member area.
Remove Access to One or More Products for an Existing User
Remove the access to one or more products for a user (by email).
