Mailtrack Integrations

Mailtrack is an email tracking extension for Gmail. With Mailtrack, you can see how people interact with the emails you send from Gmail. Track if your emails have been opened, links have been clicked, and much more!

Triggers

First Click Notification
Triggers when someone click on a tracked link for the first time.
First Open Notification
Triggers when someone open an email for the first time.
First PDF View Notification
Triggers when someone view a PDF for the first time.

Actions

Send Email
Send a tracked email
