SaleDrum Integrations

SaleDrum is an automated repricing tool for Amazon marketplace sellers

Triggers

Lost Buy Box on Amazon
Triggers when specified SKUs lost buy box on Amazon.
New Amazon Order
Triggers when a new Amazon order is received.
Amazon Low Stock
Triggers when Amazon quantity for a listing is less than a specified number.
Amazon Low Profit Margin
Triggers when profit margins of a listing go below the specified percentage.

Actions

Activate Automatic Repricing of an Amazon Channel
It allows users to activate automatic repricing of an Amazon channel
Activate a Specific Amazon Auto Repricing Rule
It allows users to activate a specific repricing rule temporarily without deleting it form associated listing
Assign a Reprice Rule to Multiple Amazon Listings
It allows users to assign a specific repricing rule to multiple Amazon Listings
Pause Automatic Repricing of an Amazon Channel
It allows users to pause automatic repricing of an Amazon channel
Pause a Specific Amazon Auto Repricing Rule
It allows users to pause a specific repricing rule temporarily without deleting it form associated listing
Remove a Reprice Rule From Multiple Amazon Listings
It allows users to remove a specific repricing rule from multiple Amazon Listings
