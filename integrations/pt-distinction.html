PT Distinction Integrations

PT Distinction enables overworked personal trainers to train more clients, take more time off, and get even better results.

Triggers

Client Added
Triggers when a new client is added.
Client Added to Group
Triggers when a client is added to a group.
Pre-Made Package Sign Up
Triggers when someone signs up to a pre-made package.

Actions

Add Client
Adds a new client. If a client with this email address has already been added it does nothing new.
Add Package to Client
Adds a Pre-Made Package to a clients account. Note that if the client isn't already set up you will need to use the Add Client action first.
Add Client to Group
Adds a client to a group.
Put Client Online
Sets the chosen client's account to online, giving the client online access.
Put Client Print Only
Sets the chosen client's account to print only.
