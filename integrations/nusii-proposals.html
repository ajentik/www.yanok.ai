Nusii Proposals Integrations

Nusii is an online proposal tool that helps you create, track and manage proposals for your creative business.

Triggers

Client Created
Triggers when a client is created.
Proposal Accepted
Triggers when a proposal is accepted.
Proposal Viewed
Triggers when a client views a proposal. Be aware that this trigger only works for Active Proposals with status: pending.
Proposal Charged
Triggers when a proposal is charged.
Proposal Created
Triggers when a proposal is created.
Proposal Rejected
Triggers when a proposal is rejected.
Proposal Sent
Triggers when a proposal is sent.

Actions

Create Client
Creates a new client.
Create Draft Proposal
Creates a new proposal.
