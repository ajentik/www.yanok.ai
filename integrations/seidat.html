Seidat Integrations

Seidat is a sales deck platform for business teams. It has tools for slide deck creation, presenting, sharing and analyzing.

Triggers

New Presentation
Triggers when a new presentation is created.
Shared Link Viewed
Triggers when a shared link is viewed.

Actions

Create Shared Link
Creates a new shared link for the presentation.
Duplicate Presentation
Duplicates an existing presentation. Useful for creating presentations from a template.
Create Presentation
Creates a new empty presentation.
Find and Replace Text in Presentation
Updates an existing presentation by finding and replacing text on the slides.
