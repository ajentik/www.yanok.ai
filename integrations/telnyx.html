Telnyx Integrations

Telnyx is a full-featured global communications platform with an SMS API built for reliable, enterprise-grade messaging on a secure network.

Triggers

New Phone Number
Triggers on a new order of a phone number
New Recording
Triggers when a new recording has been made
Receive a Call
Triggers on an incoming call to a Call Control Application
Receive a Fax
Triggers when a PDF has been received via through a fax profile
Receive a Message
Triggers when an inbound message is received by one of your Telnyx messaging profiles.
Message Status Updated
Triggers when a delivery report is received for an outbound message. Status values are "delivered", "sending_failed", "delivery_failed".

Actions

Dial a Number
Dial a Phone Number using a Telnyx Call Profile
Play Audio
Play an audio recording on an answered call
Send a Fax
Send PDF using Telnyx Fax API
Send SMS
Send an SMS message to a phone number.
Send MMS
Send an MMS message to a phone number.
Send Short Code SMS
Sends a short code message via Telnyx
Speak Text
Convert Text to Audio on a Call
