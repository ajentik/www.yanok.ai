Disciple Integrations

Disciple empowers brands and creators to build independent communities of true value. We encourage hosts to build their communities on a platform that champions members and their own values and mission.

Triggers

New Comment
Triggers when a new comment is published.
User Updated
Triggers when an existing user changes his profile details.
User Joined Group
Triggers when a user joins a group.
New Post
Triggers when a new post is published.
Product Changed
Triggers when a user's product changes
User Left Group
Triggers when a user leaves a group.
Subscription Became Active
Triggers when a user's subscription becomes active
Subscription Became Expired
Triggers when a user's subscription becomes expired
User Deleted
Triggers when a user account is deleted.
New User
Triggers when a new user registers.

Actions

Create Group Member
Adds the requested User to the Group as its new Member
Remove Group Member
Requests a removal of the requested User from the Group as its Member
Create Invite Link
Create a new Invite Link in your community.
Create Post
Create a new Post in your community.
Create Subscription Preregistration
Preregister a subscription for an email address to be assigned once the user registers.
Add Badge to User
Adds the requested Badge to the User
Remove Badge From User
Removes the requested Badge from the User
Create User Subscription
Assigns the requested Subscription to the User
Expire User Subscription
Expires the requested Subscription for the User
