Salesmsg Integrations

Salesmsg is a two-way text messaging software to allows you to send and receive text messages online from real local phone numbers.

Triggers

New Contact Opted-Out
Triggers when a contact has opted-out.
New Text Message Received
Triggers when a new text message is received.
New Text Message Sent
Triggers when a new text message is sent.
New Contact
Triggers when you add a new contact.
Missed Incoming Phone Call
Triggers when an incoming phone call is missed.
New Phone Call Received
Triggers when a phone call is received.
Tag Added to Contact
Triggers when a tag gets added to a contact

Actions

Add Tag to Contact
Applies a tag to an existing contact.
Create Note
Creates a new note on a contact record.
Create or Update Contact
Creates a contact. Can be used to update a contact too.
Send a Group Text Message (SMS / MMS)
Creates and sends a group SMS to a group of contacts. (All contacts will see each others messages.)
Phone Number Lookup (PhoneCheckr)
Identifies the number's carrier and what type of phone it is (landline, mobile, VoIP, or null).
Remove Tag From Contact
Removes a tag from a contact if they have it.
Send a Text Message (SMS / MMS)
Sends a new SMS or MMS text message.
Opt-Out a Contact
Opt-out a contact.
Find Contact
Finds a person by first name, last name, email, or phone number.
Find Salesmsg Member
Finds a Salesmsg user by first name, last name, email, or phone number.
Find Contact
Finds a person by first name, last name, email, or phone number.
