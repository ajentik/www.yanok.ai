Reamaze Integrations

Reamaze is an all-in-one helpdesk for chat, social media, SMS, and email right from your site or app.

Triggers

New Contact
Triggered when a new contact is created.
New Conversation
Triggered when a new conversation is created.
New Message
Triggered when a new message is created.

Actions

Add Message to Conversation
Adds a new message to an existing conversation.
Create Contact
Create a Contact in Reamaze.
Create SMS Contact
Create a Contact in Reamaze with an SMS mobile number.
Create Conversation
Create a new conversation on behalf of a user.
Create SMS Conversation
Create a new SMS conversation on behalf of a user.
