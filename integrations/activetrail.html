ActiveTrail Integrations

ActiveTrail is a marketing automation platform designed to help you maintain active relationships with customers, prospects, and user communities.

Triggers

Contact Added to Group
Triggers when a contact is added to a group (any group).
Contact Added via Landing Page
Triggers when a contact is added via a landing page.
Contact Added via Signup Form
Triggers when a contact is added via a signup form.
Contact Removed From Group
Triggers when a contact is removed from a group (any group).
Contact Unsubscribed
Triggers when a contact unsubscribes from receiving further emails.
Contact Updated
Triggers when a contact record in ActiveTrail is updated via the API (status or details).

Actions

Add Contact to Group
Add a new or an existing contact to a specific group (The new contact will be created if not already exists).
Remove Contact From Group
Removes a contact (passed as a parameter via the API) from a specific group (specified by the creator of the Zap).
Create/Update Contact
Creates a new contact or updates a contact if a contact already exists.
Send SMS Message
Send an operational SMS message to a mobile phone number. Notice that not all countries are supported.
Find Contact
Find an existing contact in your account.
Find Group
Find a group matching the submitted search text.
