GetAccept Integrations

GetAccept is a cloud solution where you design, send, track and promote your proposals to win and get more deals signed digitally.

Triggers

Document Commented
Triggers when a document receive a new comment
New Document
Triggers when a new document is created.
Document Rejected
Triggers when a document has been rejected by any recipient
Document Reviewed
Triggers when a document has been reviewed by a recipient.
Document Sent
Triggers when a document has been sent out to recipients.
Document Signed
Triggers when a document has been signed by all signing parties.
Document Viewed
Triggers when a document is viewed/opened for the first time by a recipient.

Actions

Archive Document
Upload a signed document to the archive or contract management module
Create Contact
Creates a new contact to use later for document sendings.
Create Document
Creates a new document and sends for signing.
Download Document
Downloads a document.
Get Document Field List
Get fields data in list format
Seal Document
Seal a document that is currently in draft status
Create Document From Template
Creates a new document with fillable fields from a selected template and send it to one or multiple recipients for signing.
Update Document Expiration
Update the expiration date and time for a document.
Upload Document
Uploads a document.
Get Document Fields
Get fields data associated with a document.
Get Document Recipients
Get recipients data associated with a document.
