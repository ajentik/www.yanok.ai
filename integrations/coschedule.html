CoSchedule Integrations

CoSchedule is mission control for your entire marketing strategy.

Triggers

New Project
Triggers when a new piece of project is created.
New Event
Triggers when a new event is created.
New Message
Triggers when a new message is created.
New Marketing Campaign
Triggers when a new marketing campaign is created.
New Task
Triggers when a new task is created on the calendar.

Actions

Create Project
Create a project and add it to your calendar OR your Workboard.
Create Event
Create an Event and add it to your calendar.
Create Social Message
Create a Social Message and add it to your calendar.
Create Marketing Campaign
Create a Marketing Campaign and add it to your calendar.
Create Task
Create a task and add it to your calendar.
Find Team Member
Find team member by email address, name or ID.
