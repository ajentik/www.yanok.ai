Loggify Integrations

A central release note repository enabling automatic communication of app updates. Build target-group specific communication pipelines now!

Triggers

New Post
Triggers when a new update is published (drafted updates don't trigger).
New or Updated Post
Triggers when a new update in Loggify is created or or an update post is updated (drafted updates don't trigger).

Actions

Create an Update Post as Draft
Create a new drafted Update in Loggify. Refinement and publishing can then be done in Loggify's UI.
