Novi AMS Integrations

Novi AMS is association management software for organizations that use QuickBooks.

Triggers

Member Created
Triggers when a member is created.
Member Activity Created
Triggers when a member activity is created.
Order Created
Triggers when an order is created (with line item support).
Event Registration Created
Triggers when a Event Registration is created (with line item support).
Member Updated
Triggers when a member is updated.
Event Registration Updated
Triggers when a Event Registration is updated (with line item support).

Actions

Updates Member

Finds an Event Registration based on unique Id or full name. Optionally by event.
