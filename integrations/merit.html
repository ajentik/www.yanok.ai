Merit Integrations

Merit is a verified identity platform that allows organizations to issue verifiable digital credentials.

Triggers

Actions

Edit a Merit (Or Send a Merit)
If a merit can be edited for the given user (as found by email & template) edit that merit using the provided data. If no merit can be found, a new merit can be issued if desired. All required data for the given merit must be provided.
Edit a Merit (One or More Fields)
Edit one or more fields of an existing merit. Very useful in cases where you only need to edit some of the data on a merit or only have access to some of the data on the merit currently.
Revoke a Merit
Revoke a merit of the specified template from the user as designated by email.
Send a Merit
Send a merit to a user. Most efficient action if you know that you only want to send new merits.
