Brick FTP Integrations

Get your own server to send and receive files to and from your clients, vendors and co-workers. Supports any browser, operating system, or FTP program. Flexible permissions and military-strength encryption protect your files.

Triggers

New File Uploaded
Triggers when a file is uploaded to a given folder. Works best in folders with fewer than 10,000 entries.
New Group
Triggers when a new group is created.
New User
Triggers when a new user is created.

Actions

Create Group
Creates a new group.
Create Permission
Creates a new permission.
Create User
Creates a new user.
Upload File
Uploads a file.
