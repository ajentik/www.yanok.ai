Pipeline Integrations

Pipeline extends your workflows across teams, departments, and companies.

Triggers

New Ask
Triggers when a new Ask is created, closed or canceled. Note, this trigger will only occur once per status.
Update Ask
Triggers when an Ask is updated. Note, this trigger could occur multiple times for the same Ask.

Actions

Add AdHoc Step
Adds an adhoc step to the Ask for a contact or a team
Create Bundle
Creates a Bundle with you as the owner
Add Ask to Bundle
Adds an Ask to a Bundle
Attach File to Step
Attaches a file to a step in the ask
Add Follower to Ask
Adds a follower to an Ask
Add Note to Step
Adds a note to a step in the ask
Cancel Ask
Cancels an open Ask
Close Ask
Closes an open Ask
Complete Step
Completes a step in an Ask
Create Ask
Creates an Ask with you as the submitter
Create Contact
Create a contact in Pipeline
Reassign Step
Reassigns a step to someone
Remove Ask From Bundle
Removes an Ask from a Bundle
Send Email
Sends a Pipeline notification email with a customizable body
Skip Step(s)
Skips steps in an ask
Spawn Ask
Spawns an ask from a parent ask
Update Field
Update a field in an Ask
Find Ask
Finds an Ask based on search criteria
Retrieve Ask
Retrieves an Ask with full data based on the ID
