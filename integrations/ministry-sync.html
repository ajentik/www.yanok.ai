Ministry Sync Integrations

Ministry Sync is an event management software for event coordinators and participants to promote and raise funds for their cause. AttendEasy and FundEasy are front-end applications that connect with Ministry Sync.

Triggers

Event Trigger
Triggered when actions happen with an event.
Event Sponsor Trigger
Triggered when actions happen with an event sponsor.
Group Trigger
Triggered when actions happen with a group.
Payment Trigger
Triggered when actions happen with a payment.
Registrant Trigger
Triggered when actions happen with a registrant

Actions

Find Events
Finds events by id.
Find Event Sponsors
Finds event sponsors by id.
Find Groups
Finds groups by id.
Find Registrants
Finds a registrant by id.
