Postscript Integrations

Postscript is a platform for SMS marketing for Shopify stores.

Triggers

New Message
Triggers when a new message is sent to a shop.
Subscriber Opted In
Triggers when a new subscriber for a shop opts in.
Subscriber Opted Out
Triggers when a shop's subscriber opts out of text messages.

Actions

Create Subscriber
Creates a new Subscriber. Whenever you are collecting phone numbers to use for marketing purposes, like with Postscript, you need to do so compliantly. Make sure you include TCPA and CTIA-compliant language wherever you collect subscribers from. Please see our guide for more on compliance:
Send Message
Sends a Message to a Subscriber.
Update Subscriber
Updates a Subscriber.
Find Subscriber
Finds a Subscriber
