GymMaster Integrations

GymMaster is an advanced membership management system helping simplify business for thousands of fitness clubs around the world.

Canceled Class Booking
Triggers on Canceled class booking.
Canceled Membership

Canceled Service Booking
Triggers on Canceled service booking.
Executed Task
Triggers on Task activation.
Expired Membership
Triggers when a membership expires, will not trigger when cancelled.
Expired Membership Add-On
Triggers when a membership add-on expires, will not trigger when cancelled.
New Measurement
Triggers when a new measurement is made.
New Class Booking
Triggers on a new class booking.
New Class Member Enrollment
Triggers when a member is booked into a class.
New Member
Triggers on New Member.
New Membership

New Service Booking
Triggers on a new service booking.
New Staff

New Prospect
Triggers on a new prospect.
New Visitor Entry
Triggers on a new visitor_entry.

Actions

Create Personal Booking for a Member
Creates Booking for a member.
Create Measurement for a Member
Creates a measurement for a member.

Creates a prospect member. Existing prospects will be reclassified as 'new' by task triggers.
