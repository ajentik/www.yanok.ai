Sierra Interactive Integrations

Sierra Interactive is an all-in-one real estate platform that helps you close more deals and grow your business.

Triggers

New Lead
Triggers when a new lead is created in your account.
New Request for More Info
Triggers when a new "Request for More Info" about a property is created by a lead.
New Site Visit
Triggers when a lead visited site.
New Saved Listing
Triggers when a listing is saved by a lead.
New Schedule Showing Request
Triggers when a new "Schedule Showing Request" about a property is created by a lead.
New Tag
Triggers when a tag is applied to lead by a user in our admin area.
New Task
Triggers when a new task is created by a user in our admin area.

Actions

Create Note
Add a Note to the lead's history.
Apply Fully Automated Action Plan
Apply a Fully Automated Action Plan.
Apply Traditional Action Plan
Apply a Traditional Action Plan.
Create Lead
Create a new lead in your Sierra account.
Create Task
Add an upcoming task for the lead.
Find Lead
Find lead in your Sierra account.
Stop Fully Automated Action Plan (Lead Responded)
Stop the current fully automated action plan and count the lead as having responded to the plan.
Stop Fully Automated Action Plan (Other Reason)
Stop the current fully automated action plan without counting the lead as having responded to the plan.
Remove Traditional Action Plan
Remove a Traditional Action Plan.
Update Lead
Update an existing lead in your Sierra account.
