Voodoo SMS Integrations

Voodoo SMS is an SMS platform that allows you to send text messages to your customers quickly and easily.

Triggers

New Blacklisted Number
Triggers when a new number is blacklisted.
New Contact List
Triggers when a new Contact List is created.
New Inbox Message
Triggers when a new message has come into your Inbox.
SMS Sent
Triggers when an SMS has been sent.

Actions

Add to Blacklist
Adds a number to the Blacklist.
Send Bulk SMS
Send bulk SMS to multiple recipients.
Add Contact
Adds a new contact into a Contact List.
Send SMS
Send a single SMS to a single recipient.
Update Contact List
Update an existing Contact List.
Update Contact List - Static
Update an existing Contact List that requires only numbers.
