Moxo Integrations

Moxo is a OneStop Client Interaction Hub.

Triggers

Flow Updates
Triggers when a flow or any of its steps is started, completed or canceled.
Xero Purchase Order Completed
Trigger when a Xero Purchase Order is approved or declined in Moxtra.

Actions

Add User to Conversation
Add an existing user to a group conversation.
Create Client
Creates a new client user.
Create Group Conversation
Creates a group conversation and add users to it.
Create Note
Creates a Note in a conversation.
Create To-Do
Creates a To-Do in a conversation.
Create Transaction
Creates a Transaction in a conversation.
Create or Update Client
Updates an existing client or creates a new one if that client doesn't already exist.
Create Xero Purchase Order
Creates a Purchase Order in Moxtra based on a Xero 'New Purchase Order' Trigger.
Deactivate Client
Deactivates a client user.
Delete Xero Purchase Order
Deletes a Purchase Order in Moxtra based on a Xero 'New Purchase Order' with 'Deleted' status Trigger.
Invite People to Meeting
Invite anyone to a previously scheduled Meeting.
Schedule Meeting
Schedule a Moxtra Meeting and invite participants to it.
Send File
Sends a file to a conversation.
Send Message
Sends a message to a conversation.
Start Flow Conversation
Start flow conversation from a flow template.
Update Client
Updates the client info and account managers for a client.
Find Direct Conversation
Finds a direct conversation between an account manager and a client.
