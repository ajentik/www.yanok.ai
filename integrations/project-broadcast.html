Project Broadcast Integrations

Project Broadcast is a robust business texting platform that allows you to communicate with your customers with ease while maintaining that personal connection.

Triggers

Actions

Add to Campaign
Add a contact to a Project Broadcast campaign. Your campaign name must be exactly the same as it appears in Project Broadcast
Apply Keyword
Apply Keyword To Project Broadcast Contact. Your keyword should already be created in Project Broadcast.
Create Contact
Creates a new Project Broadcast contact. If the contact possibly exists, you will need to use the Find Contact action which will search for the contact and create one if it is not found.
Send Message
Send Message To Project Broadcast Contact. The message will originate from your Project Broadcast Number and will show in your Project Broadcast Chat.
Update Contact
Update Contact in Project Broadcast. The contact needs to already exist for this action.
Find Contact
Finds a contact by searching.
Find or Create Contact
Finds a contact by searching.
