Qualified.io Integrations

Qualified helps you identify the best technical candidates faster and better with Code Assessments and Pair-Programming Interviews

Triggers

Assessment Result Submitted
Triggers when a candidate submits their assessment result.
Candidate Label Added
Triggers when a label is added to a candidate record.
Candidate State Changed
Triggers when a candidate's state is changed, including when a new candidate is added to the system.
Review Submitted
Triggers when a review for an assessment result is submitted.

Actions

Add Note
Adds a new note to a candidate
Invite to Assessment
Invites a candidate/student to an assessment. This can be an existing or new email.
Update Candidate
Updates a candidate's name, labels or state
Find Assessment
Lookup an existing assessment
Find Assessment Invitation
Finds an existing assessment invitation. Will also return the assessment result if the assessment has been started already.
Find Candidate
Finds an existing candidate
Find or Create Assessment Invitation
Finds an existing assessment invitation. Will also return the assessment result if the assessment has been started already.
