Google Tables Integrations

Google Tables helps teams easily track & automate tasks, enabling them to save time and work smarter.

Triggers

New or Updated Row
Triggers when a row is created or updated in a table.
New Row
Triggers when a new row is added to a table.
New Row in View
Triggers when a new row is added to a specific view in a table.

Actions

Create Row
Creates a new row in a table.
Update Row
Updates an existing row in a table.
Find Row
Finds a row by a column and value. Returns the entire row if one is found.
Find or Create Row
Finds a row, or creates one if none are found.
