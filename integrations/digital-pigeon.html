Digital Pigeon Integrations

Digital Pigeon is a large file sharing service for digital media producers, creatives, and anyone wishing to send large files via the internet.

Triggers

List Uploads
Triggers when Uploads are created.
Upload Access Event
Triggers when an upload is accessed by a recipient.
Upload Expiring Soon Event
Triggers when an upload is reaching the expiry date allowing you to update the expiry, or to inform recipients.
Upload File Commented Event
Triggers when a comment is placed on a file within an upload.
Upload Status Changed
Triggers when an upload has moved between states such as EDITING to COMPLETED.
Recipient Notification Bounced
Triggers when there are issues sending a notification email to a recipient.
Recipient Notified
Triggers when recipients have been notified about upload changes.

Actions

Archive an Upload
Moves the upload into Archive storage
Update Expiry Date
Changes the expiry date for an upload
Restore an Upload
Moves an archived upload to standard storage
Set Download Permission for an Upload
Updates the ability to download files for an upload
Set Feedback & Approvals for an Upload
Updates the ability to provide feedback and approvals for files in an upload
Find Uploads
Finds an Upload
