E-goi Integrations

E-goi is a Multichannel Marketing Automation Platform, including email marketing services, SMS, Voice Broadcast, Push Notifications, Web Push and Forms to Capture, Automate, Communicate, Analyse and Generate more Sales.

Triggers

New Subscriber
Triggers when a new subscriber is added to a list.
New Unsubscriber
Triggers when any current subscriber unsubscribes from a list.

Actions

Add/Update Subscriber
Adds a new user to a list or Updates existing user by E-mail.
Send E-Mail Campaign
Sends an e-mail campaign to a user.
Send SMS Campaign
Sends a SMS campaign to a user.
Send SMS Transactional
Sends a SMS to a cellphone
Start Autobot
Starts an autobot on an action of your choosing.
Add Tag
Attaches a tag of your choosing to an existing user.
Remove Subscriber
Removes an existing user from a list.
Search a Subscriber or Create
Searches for the subscriber in your list and resturns his data. If not found you can choose to add him.
Search or Create a Subscriber
This will return you a found subscriptor our a just created subscritor.
