Aspose Integrations

Aspose is a document automation and reporting service (Word, PDF, OpenOffice, HTML, etc). Create, edit, convert, compare, merge, split & perform other documentation tasks with ease.

Triggers

Actions

Classify
Classifies documents in various file formats (pdf, html and others).
Classify Content
Classifies the contents of documents in various file formats (pdf, html and others).
Convert
Converts a document to various file formats (pdf, html and others).
Convert Content
Converts the contents of a document to various file formats (pdf, html and others).
Merge
Merges several documents and saves the result as a single document.
Merge Content
Merges the contents of several documents and saves the result as a single document.
Merge From Archive
Merges several documents from a ZIP archive and saves the result as a single document.
Report
Generates a report using a document template and an external data source.
Report Content
Generates a report using a document template and an external data source.
Report LINQ
Generates LINQ report using a document template and an external data source.
Report LINQ Content
Generates LINQ report using a document template and an external data source.
Split
Splits a document into separate pages.
Split Content
Splits the contents of a document into separate pages.
