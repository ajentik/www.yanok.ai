Tapfiliate Integrations

Tapfiliate is cloud based affiliate tracking software. It enables you to create, track and optimize your own affiliate programs.

Triggers

Affiliate Added to Program
Triggers when an affiliate is added to a program (i.e. signed up through your program's sign up form or was added by you). Use this trigger if you need access to the affiliate's referral link.
Affiliate Approved for Program
Triggers when an affiliate is approved for one of your programs.
New Affiliate
Triggers when a new affiliate is created (i.e. signed up or was added by you).
New Affiliate Prospect
Triggers when a new affiliate prospect is created (i.e. signed up or was added by you).
Affiliate Prospect Promoted
Triggers when a new affiliate prospect is promoted to affiliate (i.e. created a Conversion or Customer).
New Conversion
Triggers when a new conversion is created.
New Customer
Triggers when a new customer is created.
New Payment
Triggers when a new affiliate payment is created.

Actions

Add Commission to Conversion
Adds a new Commission to an existing Conversion. This is useful for recurring subscription work flows.
Create Conversion
Creates a conversion.
Create Customer
Creates a customer.
Set Affiliate Group
Sets the affiliate group for an affiliate.
Set Parent Affiliate
Sets an affiliate as the parent affiliate of another affiliate. This is useful when you are making use of Tapfiliate's Multi Level Marketing (MLM) feature.
Update Commission Approval Status
Approves or Disapproves a Commission.
Update Customer Status
Cancels or Uncancels a Customer.
Find Affiliate
Finds an affiliate.
Find Affiliate in Program
Finds an affiliate in a certain program. Searching for affiliates in a program has the benefit of the affiliates referral link being returned.
Find Affiliate Prospect
Finds an affiliate prospect.
Find Conversion
Finds a conversion by external ID.
Find Customer
Finds a Customer by Customer ID.
