Bloomerang Integrations

Bloomerang is a simple, cloud-based donor database solution that pools together the latest in best practices for donor engagement and retention.

Triggers

New Constituent
Triggers when a new constituent is created.
New Transaction Payment
Triggers for each donation, pledge payment, and recurring donation when a new transaction is created. Use this to trigger for each payment in a split payment.
New Interaction
Triggers when a new interaction is created.
New Task
Triggers when a new task is created.
New Transaction
Triggers when a new transaction is created (with line item support).
Updated Constituent
Triggers when a constituent is updated.
New or Updated Constituent
Triggers when a constituent is created or updated.

Actions

Create Constituent
Creates a new constituent.
Create Donation
Creates a new donation (with line item support, 10 max.).
Create Interaction
Creates a new interaction.
Create Task
Creates a new task.
Update Constituent
Updates an existing constituent.
Find Constituent
Find a Constituent by name and a least one of address, phone, or email.
Find or Create Constituent
Find an existing constituent, or create new constituent if one is not found.
