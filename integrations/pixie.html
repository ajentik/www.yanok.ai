Pixie Integrations

Pixie is practice management for accountants and bookkeepers. Improve visibility, collaboration and automation for your team and clients.

Triggers

Job Assigned
Triggers when a job is assigned or reassigned to a team member.
Job Finished
Triggers when all tasks of a job are completed.
Job Started
Triggers when a job starts.
Client Created/Updated
Triggers when a new client is created or updated in Pixie.
Contact Created/Updated
Triggers when a contact is created or updated.

Actions

Add/Update Contact to Client
Create or update a contact, based on the contact ID or the contact's email already existing.
Add Recurring Task to Client
Creates a recurring task using the default template recurrence settings.
Complete Job
Mark an entire Job as have been completed.
Create/Update Client
Create or update a client, based on the Client ID or the client's name already existing.
Invite New User
Sends an invitation on your behalf to a team member asking them to join a workspace on Pixie.
Start Job
Starts a new job.
Find Client
Finds an existing client based on the name or ID. If neither is provided, returns all clients.
Find Contact
Find an existing contact.
